How It Works

From installation to production in three steps.

1

Install the Core

Clone the repository. Run the setup script. The core infrastructure is ready in minutes.

dotnet new tensectra-erp -n MyERP cd MyERP dotnet run
2

Add Modules

Install pre-built modules or create your own. Each module follows the same clean architecture pattern.

dotnet add package \ Tensectra.Modules.Inventory dotnet add package \ Tensectra.Modules.HR
3

Ship to Production

Deploy to your cloud provider. The kit includes Docker files, CI/CD templates, and deployment guides.

docker build -t myerp . docker push myerp kubectl apply -f k8s/
