## 1. CODE HOSTING: GitHub
**Service:** GitHub (Free Tier)
**Cost:** $0
**Integration Model:**
- Public repository for open-source visibility
- Private repository option available
- GitHub Actions for CI/CD (free 2,000 minutes/month)
- Branch protection rules for main branch
- README.md with project documentation
- LICENSE file (MIT recommended)

**Setup Steps:**
1. Create GitHub account at github.com
2. Initialize new repository "tensectra-website"
3. Push all website files to repository
4. Enable GitHub Pages in repository settings
5. Configure custom domain (optional)

---

## 2. WEB HOSTING: Netlify
**Service:** Netlify (Starter Tier)
**Cost:** $0
**Integration Model:**
- Connect GitHub repository for auto-deployment
- Continuous deployment on every push to main
- Custom domain support with free SSL
- Branch previews for testing
- Form handling (100 submissions/month free)
- Edge CDN for global performance

**Setup Steps:**
1. Sign up at netlify.com using GitHub account
2. Click "Add new site" → "Import an existing project"
3. Select GitHub repository
4. Configure build settings (none needed for static HTML)
5. Set publish directory to root (./)
6. Deploy site

**Alternative:** GitHub Pages (also free)
- Enable in repository Settings → Pages
- Select source branch (main)
- Site deploys to username.github.io/repo-name

## INTEGRATION WORKFLOW SUMMARY

### User Journey 1: Course Enrolment
1. User visits tensectra.com/courses
2. Selects course and clicks "Enrol"
3. Fills Netlify application form
4. Form submission triggers:
   - Email notification to team
   - HubSpot contact creation (via Zapier)
   - Manual review within 48 hours
5. Acceptance email with Stripe payment link
6. Payment confirmation triggers:
   - Discord channel invite
   - Course materials access
   - Welcome email sequence

### User Journey 2: Pro Membership
1. User visits tensectra.com/pro
2. Selects monthly or annual plan
3. Stripe Payment Link checkout
4. Subscription confirmation triggers:
   - HubSpot deal creation
   - Discord Pro channel access
   - Monthly digest subscription
   - Welcome onboarding email

### User Journey 3: Corporate Consultancy
1. User visits tensectra.com/consultancy
2. Clicks "Enquire for Tier X"
3. Fills Netlify enquiry form
4. Form submission triggers:
   - Email to sales team
   - HubSpot deal pipeline entry
   - Calendly booking link sent
5. Discovery call scheduled
6. Proposal sent via HubSpot
7. Contract signed (HelloSign/DocuSign)
8. Invoice paid via Stripe
